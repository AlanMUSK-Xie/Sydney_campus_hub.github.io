# Sydney Campus Hub - 悉尼高校校园服务网站

## 项目简介
这是一个为悉尼6所高校学生提供一站式校园服务的网站，包含学术服务、二手交易、社交分享和兼职对接等功能。

## 文件结构
```
sydney-campus-hub/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # 交互脚本
└── README.md       # 说明文档
```

## 功能特点
1. **响应式设计** - 完美适配电脑端、平板和手机端
2. **简洁实用** - 无复杂动画，注重用户体验
3. **易于调整** - 使用标准CSS，方便修改样式和布局

## 使用方法

### 方式一：可视化编辑器（推荐，适合零基础）
1. 在浏览器中打开 `editor.html` 文件
2. 左侧面板可以调整各种样式参数：
   - 导航栏颜色、高度
   - Banner标题大小、颜色、内边距
   - 功能卡片间距、内边距、边框颜色
   - 底部背景色、文字颜色
3. 调整后点击"应用更改"按钮实时预览效果
4. 满意后可以点击"导出CSS"获取代码，复制到 `styles.css` 文件中

### 方式二：直接查看网站
1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 使用代码编辑器（如VS Code）可以修改样式和内容
3. 所有样式都在 `styles.css` 文件中，可以根据需要调整颜色、大小等

## 主要区域说明

### 顶部导航栏
- 背景色：浅灰色 (#e8e8e8)
- 文字颜色：深蓝色 (#1a4d7a)
- 包含5个导航按钮，移动端自动切换为汉堡菜单

### 首页Banner
- 显示网站名称和核心标语
- 包含校园场景图片（自动从Unsplash加载）

### 核心功能入口
- 4个卡片式入口，点击可跳转到对应板块
- 每个卡片包含图标、标题和描述

### 底部区域
- 显示联系方式和版权信息
- 背景色：浅灰色，文字：黑色

## 自定义修改指南

### 修改颜色
在 `styles.css` 文件中搜索以下颜色代码进行修改：
- 导航栏背景：`#e8e8e8` (浅灰色)
- 文字颜色：`#1a4d7a` (深蓝色)
- 底部背景：`#e8e8e8` (浅灰色)

### 修改图片
在 `index.html` 中找到 `<img>` 标签，修改 `src` 属性为你的图片地址

### 添加新功能
1. 在 `index.html` 中添加新的HTML结构
2. 在 `styles.css` 中添加对应的样式
3. 在 `script.js` 中添加交互逻辑（如需要）

### 部署步骤
详细步骤请查看：

1. 在GitHub上创建仓库：`sydney-campus-hub`（必须是公开仓库）

3. 在仓库Settings中启用GitHub Pages
4. 等待1-2分钟，访问网站
